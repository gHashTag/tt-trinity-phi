## Badge URLs (Shields.io)

### Template

```
https://img.shields.io/badge/{Label}-{Message}-{Color}
```

### Common Colors

- `brightgreen` = #4c1 for success
- `yellow` = #dfb317 for warning
- `red` = #e05d44 for failure
- `blue` = #3178c6 for information
- `grey` = #9e9e9e for disabled

---

## Dynamic Badges (GitHub)

For GitHub-specific status, use:

```markdown
[![](https://img.shields.io/github/actions/workflow/status/{user}/{repo}/test.yml)](https://github.com/{user}/{repo}/actions)
```

Example:

```markdown
[![](https://img.shields.io/github/actions/workflow/status/gHashTag/tt-trinity-phi/test.yml)](https://github.com/gHashTag/tt-trinity-phi/actions)
```
